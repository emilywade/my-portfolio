PROJECTS

Javascript Reaction Timer

I went from having zero Javascript knowledge, to understanding how to make Javascript interact with HTML, how to make pages behave as you want them to, and then how to create complex code structures using variables, arrays, loop and IF statements. I was then able to create this reaction tester, as shown to the left. It measures how long it takes you to click on the shape on the page once it appears. The shape appears in a random position on the page each time you play, it takes a random amount of time to appear, it's randomly sized and coloured, and it's randomly a square or a circle. This challenge was the first I completed that required a combination of HTML, CSS and Javascript, and it introduced timing and randomness.

jQuery Code Player

jQuery was taught as an introduction to Javascript frameworks, and explained why you may want to include them in your sites. I learnt how to animate and fade content, how to use jQuery and AJAX to build interactive content that doesn't require browser refreshes, and how to implement form validation. The Code Player which I built was based on the website jsbin.com. It basically serves as a way to share code in a concise and easy-to-interpret manner. I built a slightly simplifed version, which has all the core functionalities: HTML, CSS, Javascript and an Output.

Python Mastermind

A lot of the content covered in this chapter was stuff that I was already familar with from my time at University. It covered Python basics including variables, arrays, loops, IF statements and functions. The challenge at the end of the chapter was to build a version of the 2-player game Mastermind. I learnt how to interact with the user to create a guessing-game that gave feedback to the user as they guessed correctly or incorrectly.
